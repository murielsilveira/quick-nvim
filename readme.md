# Quick Neovim

Run a temporary pre-configured Neovim instance anywhere you need, quickly.

`curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/murielsilveira/quick-nvim/main/install.sh | sh`
